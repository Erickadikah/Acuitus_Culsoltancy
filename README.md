# Installation

1. Clone the repo
   ```sh
   git clone
    ```
2. Install NPM packages
    ```sh
    npm install
    ```
3. Run the app
    ```sh
    npm run dev
    ```
4. Open the app
    ```sh
    http://localhost:3000
    ```
